Instructions for running different algorithms:

You will need `docker` and the benchmark-files should be in DIMACS format. The source for each algorithm is at `/scalmc/src/scalmc.cpp`.
A prebuilt static binary already exist at `/scalmc/build/`.To rebuild the binary after changing the source run `make` from `/scalmc/build` directory.

  • SparseCount
    • docker run -it bhavishya/sparsecount bash
    /scalmc/build/scalmc `benchmark-file`
  • SparseCount2
    • docker run -it bhavishya/sparsecount2 bash
    /scalmc/build/scalmc `benchmark-file`
  • ApproxMC2
    • docker run -it bhavishya/scalmc bash
    /scalmc/build/scalmc `benchmark-file`
  • Toeplitz-ApproxMC2
    • docker run -it bhavishya/toeplitz bash
    /scalmc/build/scalmc `benchmark-file`
